## 👋 Wilfer Becerra · Founder @ [Albor Automations](https://albor-automations.com)

Construyo trabajadores digitales con IA para PYMEs en España. Voice agents que atienden llamadas 24/7, sistemas que clasifican y responden email, agentes que recuperan leads dormidos. Sin chatbots tontos. Sin transformación digital.

Si lo construyo, lo despliego en producción. Si lo despliego, mido las horas que ahorra. Si no las ahorra, lo replanteo.

<br />

## 🛠️ Stack

```
Languages   →  TypeScript  ·  Python  ·  Bash
Frontend    →  Next.js  ·  React  ·  Tailwind  ·  shadcn/ui
Backend     →  Node  ·  NestJS  ·  Express  ·  Edge Runtime
AI          →  Claude API  ·  GPT  ·  Pinecone  ·  Supabase
```
